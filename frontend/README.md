# Phase 1 - Real-Time Dashboard

## Quick Start

1. Install Flask:
```bash
pip install flask flask-cors pymongo
```

2. Start Flask server:
```bash
python server.py
```

3. Open dashboard:
```bash
open index.html
```

## Files

- `index.html` - Dashboard HTML
- `styles.css` - Styling
- `app.js` - Frontend JavaScript
- `server.py` - Flask API server
- `README.md` - This file

## Features

- Real-time sentiment distribution
- Trending entities
- Language breakdown
- Active topics
- Alert feed
- Live post stream
- Dashboard stats

Updates every 5 seconds automatically.
